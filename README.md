# z3

Crystal API for Z3 Theorem Prover.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     z3:
       github: taw/crystal-z3
   ```

2. Run `shards install`

## Usage

```crystal
require "z3"
```

See `examples` folder for some examples.
